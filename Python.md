# Console-based-booking-system
Developing a console-based booking system named DropMe™ for the imaginary island nation.
This program allows users to inquire about traveling fares between cities, provide promotion discounts and generate invoices automatically. This program incorporates major programming aspects such as modules, functions, file handling, error handling and randomization.

calculates minimum traveling fares between five major cities and allows users to select vehicle types Trishaw (default), Car or Van with price multipliers based on each.
